# Changelog

All notable changes to Musical Groove are documented here.

## v1.0.0 — 2026-06-28

首个公开版本 / First public release.

- **全屏音乐可视化 / Fullscreen visualizer**：弯曲穹顶上的方块柱场随系统声音律动，中心发光反应堆核心；连续发光地面，深色场域 + 高峰对比。
- **系统声音捕获 / System audio**：macOS ScreenCaptureKit 系统回环（免装 BlackHole），需屏幕录制权限。
- **正在播放 / Now playing**：顶部歌曲卡（封面 / 曲名 / 艺人），可展开拖动进度、上一首 · 暂停 · 下一首；自动避开刘海屏。
- **同步歌词 / Synced lyrics**：来自 lrclib，按曲名/艺人匹配；封面主色微调场景配色。
- **待机模式 / Standby**：静音时柔和的待机波纹 + 偶发流星。
- **控制台 / Console**：点 LyHN 签名打开；45 项参数 + 版本预设 + 默认风格；不保存关闭自动还原到调整前。
- **打包 / Packaging**：macOS Apple Silicon (arm64) dmg，未签名 / ad‑hoc。
